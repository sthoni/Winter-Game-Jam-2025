# Winter Game Jam 2025

## Idee

In einer Arena muss der Spieler die Gegner einfrieren, um sie anschließend zerschmettern zu können.
Er bewegt sich dabei ähnlich wie Megaman, verfügt zusäzlich aber auch über einen Dash. Dieser Dash sorgt auch dafür, dass Gegner zerschmettert werden.
Die Gegner sind Flammenwesen. Wenn man sie berührt oder durch sie dasht, bekommt man Schaden. Bestimmte Gegnertypen schießen auch Flammenbälle ab. Diese kann man auch mit der Gefrier-Waffe abwehren. Das Spiel wird durch einen Bosskampf beendet.
Als Ziel des Jams könnte es sein, dass es entweder Punkte für Zeit und verschossene Projektile oder so gibt.
